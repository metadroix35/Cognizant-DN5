Exercise 3 - Stored Procedures

Objective
---------
Learn how to create and execute stored procedures in PL/SQL to automate banking operations.

Scenarios Implemented
---------------------
1. ProcessMonthlyInterest
   Applies a 1% monthly interest to all savings accounts.

2. UpdateEmployeeBonus
   Updates employee salaries in a specified department by adding a bonus percentage.

3. TransferFunds
   Transfers money between two accounts after checking that the source account has sufficient balance.

PL/SQL Concepts Used
--------------------
- CREATE PROCEDURE
- Parameters
- UPDATE
- SELECT INTO
- IF Statement
- RAISE_APPLICATION_ERROR
- EXCEPTION Handling
- COMMIT
- ROLLBACK
- DBMS_OUTPUT.PUT_LINE

Learning Outcomes
-----------------
- Created reusable stored procedures.
- Passed parameters to procedures.
- Performed conditional checks.
- Managed transactions using COMMIT and ROLLBACK.
- Implemented business logic inside stored procedures.
