Exercise 1 - Control Structures

Objective
---------
Learn the use of PL/SQL control structures such as loops, conditional processing, and DBMS_OUTPUT to automate banking operations.

Scenarios Implemented
---------------------
1. Applied a 1% discount to loan interest rates for customers older than 60 years.
2. Updated VIP status for customers having a balance greater than 10000.
3. Printed reminder messages for loans due within the next 30 days.

PL/SQL Concepts Used
--------------------
- BEGIN...END block
- FOR LOOP
- UPDATE statement
- SELECT statement
- JOIN
- COMMIT
- DBMS_OUTPUT.PUT_LINE

Learning Outcomes
-----------------
- Understood PL/SQL block structure.
- Learned to use FOR loops.
- Performed record updates using UPDATE.
- Retrieved records using SELECT and JOIN.
- Displayed output using DBMS_OUTPUT.
